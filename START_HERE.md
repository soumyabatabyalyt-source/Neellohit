**A:**

2. Check the relevant guide

## 🗺️ The Big Picture

Here's what you're building:

```
Your App
├─ Frontend (What users see)
│  ├─ Login page
│  ├─ Signup page
│  ├─ Pending approval page
│  └─ Dashboard
│
├─ Backend (Hidden logic)
│  ├─ Signup endpoint
│  ├─ Login check
│  └─ Approval logic
│
└─ Database (Stored data)
   └─ User profiles with approval status
```

**The approval system:**
- Users can't login if `approved: false`
- Managers approve them (set `approved: true`)
- Then they can login!

**That's it! Everything else is details.**
